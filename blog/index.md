Welcome to the practical guide for Scala developers who want to learn (or re-learn) Java!

This blog covers modern Java features, comparing them with their Scala equivalents, and providing practical examples to help you transition between the two languages.

## Modules

This repository contains examples in multiple languages:

- **[Java 21](https://github.com/sps23/java-for-scala-devs/tree/main/java21)** - Modern Java with preview features
- **[Scala 2](https://github.com/sps23/java-for-scala-devs/tree/main/scala2)** - Scala 2.13 examples
- **[Scala 3](https://github.com/sps23/java-for-scala-devs/tree/main/scala3)** - Scala 3 examples
- **[Kotlin](https://github.com/sps23/java-for-scala-devs/tree/main/kotlin)** - Kotlin examples for comparison
