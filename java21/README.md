# Java 21 Module

This module contains Java 21 examples for the Java for Scala Developers project, including preview features.

## Package

All code is organized under the `io.github.sps23` package.

## Build

```bash
./gradlew :java21:build
```

## Test

```bash
./gradlew :java21:test
```

## Features

This module demonstrates Java 21 features such as:
- Virtual threads
- Pattern matching
- Record patterns
- String templates (preview)
- Sequenced collections
